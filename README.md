# Smart Road Accident Detection System

An AI-powered Smart Road Accident Detection System that detects road accidents and predicts accident severity using Deep Learning and Computer Vision. The system helps improve road safety by enabling faster accident identification and supporting emergency response.

## 🚀 Features

- 🚗 Road accident detection using Deep Learning
- ⚠️ Accident severity prediction
- 📷 Image-based accident analysis
- 🧠 TensorFlow/Keras trained models
- 🌐 Flask-based web application
- 📊 User-friendly interface
- ⚡ Fast and accurate predictions

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Smart-Road-Accident-Detection-System.git
```

Go to the project directory:

```bash
cd Smart-Road-Accident-Detection-System
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
